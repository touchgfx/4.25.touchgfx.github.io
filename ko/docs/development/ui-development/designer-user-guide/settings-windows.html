설정 창

TouchGFX Designer의 전역 설정은 로비 파일 메뉴를 통해 사용이 가능합니다.

위치 설정​

위치 설정을 통해 추가 정보와 함께 Live Callouts 및 Update Banner를 제공할 수 있으며,이 정보는 사용자에게 맞춤형 경험을 제공하는 데 사용됩니다.

위치 설정 창

프록시 설정​

규칙이 엄격한 네트워크의 사용자들은 프록시 설정을 통해 TouchGFX Designer의 인터넷 접속에 필요한 프록시 유형을 지정할 수 있습니다. 세 가지 유형의 프록시 구성을 사용할 수 있습니다.

Use System Proxy
현재 사용자 컴퓨터에서 사용 중인 프록시를 사용합니다. 컴퓨터에서 사용하는 프록시에서 필요한 경우에 여기에서 자격 증명을 제공할 수 있습니다.

Manual Proxy Configuration
TouchGFX Designer에서 사용할 프록시를 수동으로 지정할 수 있습니다. 컴퓨터에서 사용하는 프록시에서 필요한 경우에 여기에서 자격 증명을 제공할 수 있습니다.

No Proxy
사용자 컴퓨터의 프록시 캐시를 사용합니다.
