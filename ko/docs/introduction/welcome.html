TouchGFX 설명서

TouchGFX의 공식 설명서에 오신 것을 환영합니다!

본 설명서와 TouchGFX를 처음 접하시는 분들은 전반적인 이해를 돕기 위해 읽어보시기 바랍니다. 사이드바의 목차에서 관심 있는 주제에 대한 내용을 쉽게 찾으실 수 있습니다. 오른쪽 상단의 검색 기능을 사용할 수 있습니다.

설명서 소개​

사이트에 대한 이 주요 설명서는 다음과 같은 섹션으로 구성되어 있습니다:

  • 소개(Introduction)  - TouchGFX 및 설치 가이드에 대한 기본 정보 제공.
  • 기본 개념(Basic Concepts)  - 그래픽 주요 개념 소개.
  • 개발(Development)  - 구조, 워크플로, 도구 등을 포함한 TouchGFX 애플리케이션 개발 방법.

또한 API 및 TouchGFX 아카데미에는 상단에 있는 탭을 통해 액세스할 수 있습니다.

  • API - 클래스와 전역 변수가 포함된 TouchGFX API.
  • TouchGFX 아카데미 - TouchGFX 특징 및 기능 설명 및 소개.

대상 사용자​

TouchGFX 설명서는 TouchGFX 설명서는 C++과 STM32에서의 임베디드 GUI 개발에 관한 기본적인 스킬을 갖춘 소프트웨어 개발자를 대상으로 합니다. 임베디드 GUI 개발이 처음이신 분들은 Basic Concepts 섹션을 참조하시면 됩니다. 또한 단계별 가이드와 튜토리얼이 제공되기 때문에 누구나 TouchGFX 개발 방법을 손쉽게 배울 수 있습니다.

TouchGFX는 설명서의 내용을 개선해 나가고자 합니다.  설명서에서 이해가 되지 않는 부분이 있거나 찾는 내용이 없는 경우에는 포럼!에 참여하시어 설명서를 개선하는 데 도움을 주시기 바랍니다!
