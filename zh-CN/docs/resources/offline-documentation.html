离线文档

Progressive Web App（PWA）​

文档可以作为Progressive Web App安装。 这意味着，诸如Google Chrome和Microsoft Edge等PC浏览器提供了在本地安装文档的选项，以便在离线和联机时均可使用。

如何安装TouchGFX文档PWA​

下图显示了在Google Chrome和Microsoft Edge中可以找到PWA安装按钮的位置。

在Google Chrome中，安装按钮位于URL栏的右侧

在Microsoft Edge中，安装按钮位于URL栏的右侧

安装PWA后，打开并导航至左侧栏中的另一页（例如“What is TouchGFX？”）。 这对于触发离线缓存非常重要，当系统出现错误，在没有internet连接的情况下，可用离线缓存。
