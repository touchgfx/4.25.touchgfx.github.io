设置窗口

TouchGFX Designer's global settings are available through the Lobby File Menu.

位置设置​

位置设置可向实时标注和更新横幅提供附加信息，为用户提供量身定制的体验。

位置设置窗口

代理设置​

代理设置将允许用户在具有严格规则的网络下指定TouchGFX 设计器连接到互联网所需的代理类型。 有三种类型的代理配置可供用户使用。

使用系统代理
使用用户计算机当前正在使用的代理。 如计算机使用的代理需要，可在此处提供凭据。

Manual Proxy Configuration
Allows for manually specifying a proxy to be used by TouchGFX Designer. 如计算机使用的代理需要，可在此处提供凭据。

无代理
将使用用户机器的代理缓存。
