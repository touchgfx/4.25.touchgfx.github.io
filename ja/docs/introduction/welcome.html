TouchGFXドキュメント

本サイトでは、TouchGFXツールおよび使用方法をご紹介します。

TouchGFXを初めて使用する場合には、本ドキュメントを事前にご確認いただき概要を把握することをお勧めします。 左サイドの目次を使用して、ご関心のあるトピックのドキュメントに簡単にアクセスできます。 右上隅にある検索機能もご使用ください。

本ドキュメントについて​

本サイトは次のセクションで構成されています。

  • はじめに：TouchGFXとインストール･ガイドの概要
  • 基本概念：主要なグラフィックス概念
  • 開発：構造やワークフロー、ツールを含むTouchGFXアプリケーションの開発方法

さらに、APIとTouchGFX Academyには、上部のタブからアクセスできます。

  • API - TouchGFX APIのクラスとグローバル
  • TouchGFX Academy - TouchGFXの特徴と機能の説明と紹介

対象ユーザ​

TouchGFXドキュメントは、C++およびSTM32上の組込みGUI開発に関する基本的な技能を持つソフトウェア開発者を対象としています。 組込みGUI開発の初心者の方向けに、「基本概念」セクションでサポートしています。また、TouchGFXの開発において誰もが円滑に学習できるように、段階を追ったガイドやチュートリアルを用意しています。

より良い開発ツールやサービスをご提供できるように、弊社では本ドキュメントの改善を実施しています。 理解できない場合、あるいは探している内容が本ドキュメントに存在しない場合は、フォーラムでお知らせください。本ドキュメントの改善に繋げます。
