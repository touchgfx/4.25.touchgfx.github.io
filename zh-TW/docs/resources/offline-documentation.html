離線文件

漸進式網路應用程式(PWA)​

文件可安裝為漸進式網路應用程式。 這表示Google Chrome及Microsoft Edge等個人電腦瀏覽器，可提供選項在本機安裝文件，使其在離線及線上都可使用。

如何安裝TouchGFX文件PWA​

以下圖像顯示PWA安裝按鈕位在Google Chrome及Microsoft Edge的什麼地方。

Google Chrome的安裝按鈕位在URL列上的右側

Microsoft Edge的安裝按鈕位在URL列中的右側

安裝PWA後，請開啟並導覽至左側列的另一個頁面(例如「什麼是TouchGFX？」)。 這對觸發離線快取相當重要，這樣就會因為系統中的錯誤，在沒有網際網路連線的情況下提供使用。

離線PDF​

文件也可利用PDF格式下載。 此項PDF內容可能無法完全反映在瀏覽器的相同體驗。

您可在此找到文件的離線PDF版本。
